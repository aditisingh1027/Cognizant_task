# Arrays – Data Structures

This folder contains classic array algorithm examples implemented in C#.

- **LargestElement.cs** – Finds the maximum value in an integer array.
- **ReverseArray.cs** – Reverses an array in‑place.
- **RotateArray.cs** – Rotates an array to the right by *k* positions.

Each file includes a static method and a simple console demo (`Program.cs` can be added later).
