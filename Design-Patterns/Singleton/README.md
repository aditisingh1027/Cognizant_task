# README for Singleton Pattern

The **Singleton** pattern ensures that a class has only one instance and provides a global point of access to it. This is useful when exactly one object is needed to coordinate actions across the system (e.g., configuration manager, logger).

## Files
- `Singleton.cs` – Implementation of the thread‑safe singleton.
- `Program.cs` – Simple console demo showing how to retrieve the instance and use it.
