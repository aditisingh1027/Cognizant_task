# Week 1 – Engineering Concepts (.NET FSE)

This repository contains the work completed during Week 1 of the Cognizant Digital Nurture 5.0 .NET Full Stack Engineer learning program.

The focus of this week was understanding core software engineering concepts and implementing them through hands‑on exercises in C#. The repository includes examples of SOLID principles, commonly used design patterns, data structures and algorithms, complexity analysis, advanced SQL concepts, and unit testing.

---

## Topics Covered

### SOLID Principles

Examples and explanations for:

* Single Responsibility Principle (SRP)
* Open/Closed Principle (OCP)
* Liskov Substitution Principle (LSP)
* Interface Segregation Principle (ISP)
* Dependency Inversion Principle (DIP)

---

### Design Patterns

Implemented the following design patterns:

* Singleton
* Factory
* Builder
* Adapter
* Observer
* Strategy
* Command

Each pattern includes:

* Source code implementation
* Demo/example usage
* README with explanation

---

### Data Structures & Algorithms

Implemented and documented:

* Linear Search
* Binary Search
* Bubble Sort
* Merge Sort
* Quick Sort
* Linked List
* Complexity Analysis

Each topic includes code examples along with a short explanation of the approach and time complexity.

---

### Advanced SQL

Covered the following SQL concepts:

* Joins
* Subqueries
* Views
* Stored Procedures
* Transactions
* Indexes

Example queries and explanations are included for each topic.

---

### Unit Testing

Basic unit testing has been implemented using:

* NUnit
* Moq

Tests have been added for selected design patterns and DSA implementations to understand the fundamentals of automated testing.

## How to Run

Restore packages:

```bash
dotnet restore Week1EngineeringConcepts.sln
```

Build the solution:

```bash
dotnet build Week1EngineeringConcepts.sln
```

Run tests:

```bash
dotnet test Week1EngineeringConcepts.sln
```
